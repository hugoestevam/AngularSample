Conteúdo

Esse template vem com um sistema de gerência de Eventos. Os Eventos não são realmente pegos de uma API nem salvos nela, contudo a alteração para isso acontecer é mínima.

Eventos são relacionados à eventos de impressoras, por exemplo:

  • Faltou papel
  • Acabou tinta
  • Sem comunicação
  • Etc..

Um evento é formado por:

  • Prioridade
  • Descrição
  • Nome da impressora a qual pertence
  • Tipo do evento (Se foi gerado pelo sistema ou pelo usuário)
  • Data de ocorrência
